## 每週詳細任務

### 第 1 週：Python、pandas 與金融資料

教材：B Mini-course 1、E。

- [ ] 安裝 Python、Jupyter/VS Code、pandas、NumPy、Matplotlib、scikit-learn、statsmodels
- [ ] 建立 Git repository 與資料夾結構
- [ ] 讀取 OHLCV 資料
- [ ] 轉換日期、排序、檢查缺失值
- [ ] 計算簡單報酬與對數報酬
- [ ] 畫價格圖與累積報酬圖
- [ ] 產出：`week01_data_loading.ipynb`、`week01_report.md`

### 第 2 週：報酬、波動率與回撤

教材：A、E。

- [ ] 寫年化報酬函數

- [ ] 寫最大回撤函數
- [ ] 比較至少 5 個 ETF
- [ ] 畫資產曲線、回撤曲線與滾動波動率
- [ ] 產出績效比較報告

### 第 3 週：時間序列與資料清理
